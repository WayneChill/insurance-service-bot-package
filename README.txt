保險業務發展小幫手 LINE Bot - 安裝說明
================================================================

【步驟一：填入設定】
打開 config.txt，填入以下資訊：
- LINE_CHANNEL_ACCESS_TOKEN（LINE Developers Console）
- LINE_CHANNEL_SECRET（LINE Developers Console）
- LINE_USER_ID（你自己的 LINE User ID，接收每日早報）
- GOOGLE_SHEET_ID（Google Sheets 網址中間那串 ID）
- DRIVE_FOLDER_ID（Google Drive 資料夾 ID）
- LICENSE_KEY（向管理員取得）

【步驟二：放入憑證】
將 Google 服務帳號的 credentials.json 放入本資料夾
並確認已將服務帳號 email 加入 Google Sheets 共用（編輯者）
以及 Google Drive 資料夾共用（編輯者）

【步驟三：部署到 Railway】
1. 將本資料夾推送到你的 GitHub repo
2. Railway 連結該 repo，設定環境變數
3. 部署完成後 Webhook URL 填入 LINE Developers Console
